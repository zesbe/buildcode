üö® MASALAH: Mengapa AI Menciptakan Folder Tambahan?
1

Context Confusion

AI menerima request yang ambigu atau context dari percakapan sebelumnya yang tercampur dengan project lain

2

Template Generation

AI secara otomatis generate template project berdasarkan keyword yang terdeteksi (misal: "billing", "system")

3

Working Directory Mix

AI tidak mem-verify working directory yang benar sebelum membuat file/folder baru

4

Session Memory

Memory dari session sebelumnya atau parallel task bisa mempengaruhi output current session

‚úÖ SOLUSI: Cara Mencegah & Optimasi Maksimal
1

Clear Context

Selalu berikan context yang jelas: "Edit file X di project chat-ai saja, jangan buat project baru"

2

Directory Lock

Specify working directory di awal: "Bekerja hanya di folder C:\chat-ai-percobaan"

3

Verification Step

Minta AI untuk verify struktur folder sebelum dan sesudah perubahan

4

Git Integration

Gunakan git status untuk monitor perubahan dan rollback jika ada yang tidak diinginkan
